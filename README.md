# Nexus – Investor & Entrepreneur Collaboration Platform

## Overview

Nexus is a modern web platform designed to connect **investors and entrepreneurs** in one place.
It provides tools for **meeting scheduling, video collaboration, document management, and mock payment workflows**.

This project was developed as part of a **Frontend Internship Task** using **React.js** with a focus on clean UI, modular components, and responsive design.

---

## Features

### Week 1 – Scheduling & Setup

* Project setup and UI theme configuration
* Responsive layout with consistent colors and typography
* **Meeting Scheduling Calendar**

  * Select and manage availability
  * View scheduled meeting dates
  * Interactive calendar interface

### Week 2 – Collaboration Tools

* **Video Call Module**

  * Start / End call UI
  * Mock video interface
  * Audio / video toggle design

* **Document Chamber**

  * Upload PDF or document files
  * Preview uploaded documents
  * Contract status labels (Draft / In Review / Signed)

### Week 3 – Payments & Security

* **Payment Simulation**

  * Wallet balance display
  * Transaction history table
  * Deposit / Transfer simulation

* **Security Enhancements**

  * Login form validation
  * Password handling
  * Role-based dashboard UI concept

## Installation & Setup

Clone the repository

```
git clone https://github.com/yourusername/nexus-platform.git
```

Navigate into the project folder

```
cd nexus-platform
```

Install dependencies

```
npm install
```

Start the development server

```
npm start
```

The application will run at

```
http://localhost:3000
```
